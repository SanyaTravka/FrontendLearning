# Связующие строки

Данные также могут охватывать несколько строк, используя атрибут rowspan .

Атрибут rowspan используется для данных, охватывающих несколько строк (возможно, в определенный день событие продолжается несколько часов). Он принимает целое число (большее или равное 1) для обозначения количества строк, которые оно охватывает.

 < таблица >
  < tr > <! - Ряд 1 ->
    < th > </ th >
    < th > суббота </ th >
    < th > воскресенье </ th >
  </ tr >
  < tr > <! - Ряд 2 ->
    < th > Утро </ th >
    < td rowspan = "2" > Работа </ td >
    < td rowspan = "3" > Расслабьтесь </ td >
  </ tr >
  < tr > <! - Ряд 3 ->
    < th > день </ th >
  </ tr >
  < tr > <! - Ряд 4 ->
    < th > Вечер </ th >
    < td > Ужин </ td >
  </ tr >
</ таблица >

В приведенном выше примере четыре строки:

  1. Первая строка содержит пустую ячейку и два заголовка столбцов.
  2. Вторая строка содержит заголовок строки « Morning вместе с заголовком « Work , который занимает две строки под столбцом « Saturday . Запись «Расслабьтесь» занимает три строки под столбцом « Sunday .
  3. Третья строка содержит только заголовок строки Afternoon .
  4. Четвертая строка содержит только запись « Dinner , поскольку «Расслабление» занимает соседнюю ячейку.

Если вы хотите увидеть, как браузер интерпретирует приведенный выше код, скопируйте его и вставьте в редактор кода, чтобы лучше понять его.
