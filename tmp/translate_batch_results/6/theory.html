# Границы стола

До сих пор созданные вами таблицы было немного трудно читать, потому что у них нет границ.

В более старых версиях HTML границу можно было добавить к таблице с помощью атрибута border присвоением ему целого числа. Это целое число будет представлять толщину границы.

 < table border = "1" >
  < tr >
    < td > 73 </ td >
    < td > 81 </ td >
  </ tr >
</ таблица >

Код в приведенном выше примере устарел , поэтому не используйте его. Он предназначен для иллюстрации старых соглашений, с которыми вы можете столкнуться при чтении кода других разработчиков.

Браузер, скорее всего, по-прежнему будет правильно интерпретировать ваш код, если вы используете атрибут border , но это не означает, что атрибут следует использовать.

Мы используем CSS для добавления стиля в HTML-документы, потому что это помогает нам отделить структуру страницы от внешнего вида. Вы можете узнать больше о CSS в наших курсах CSS.

Вы можете добиться того же эффекта границы таблицы, используя CSS.

 table , td {
  граница : сплошной черный 1px ;
}

Код в приведенном выше примере использует CSS вместо HTML для отображения границ таблицы.
