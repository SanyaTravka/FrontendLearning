# Табличные данные

Для добавления данных в таблицу недостаточно строк. Также должен быть определен каждый элемент ячейки. В HTML вы можете добавлять данные с помощью элемента данных таблицы : <td> .

 < таблица >
  < tr >
    < td > 73 </ td >
    < td > 81 </ td >
  </ tr >
</ таблица >

В приведенном выше примере две точки данных ( 73 и 81 ) были введены в одну существующую строку. Добавив две точки данных, мы создали две ячейки данных.

Если бы таблица отображалась в браузере, она показывала бы таблицу с одной строкой и двумя столбцами.
