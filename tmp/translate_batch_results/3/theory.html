# Строки таблицы

Во многих программах, использующих таблицы, таблица уже предопределена для вас, что означает, что она содержит строки, столбцы и ячейки, в которых будут храниться данные . В HTML все эти компоненты должны быть созданы.

Первым шагом при вводе данных в таблицу является добавление строк с помощью элемента строки таблицы : <tr> .

 < таблица >
  < tr >
  </ tr >
  < tr >
  </ tr >
</ таблица >

В приведенном выше примере в таблицу были добавлены две строки.
