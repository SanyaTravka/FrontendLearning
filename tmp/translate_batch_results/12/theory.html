# Стилизация с помощью CSS

Таблицы по умолчанию очень мягкие. У них нет границ, цвет шрифта черный, а шрифт того же типа, что и для других элементов HTML.

CSS , или каскадные таблицы стилей, - это язык, который веб-разработчики используют для стилизации HTML-содержимого на веб-странице. Вы можете использовать CSS для стилизации таблиц. В частности, вы можете стилизовать различные аспекты, упомянутые выше.

 table , th , td {
  граница : сплошной черный 1px ;
  семейство шрифтов : Arial , sans-serif ;
  выравнивание текста : центр ;
}

Код в приведенном выше примере демонстрирует лишь некоторые из различных аспектов таблицы, которые можно стилизовать с помощью свойств CSS.
